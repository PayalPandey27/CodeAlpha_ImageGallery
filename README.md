# CodeAlpha_ImageGallery
A responsive image gallery built using HTML,CSS,enhanced with JavaScript functionality for navigation and lightbox  viewing.Users can browse images, filter them by country, and view them in a fullscreen lightbox with keyboard navigation.

#Features
- Responsive grid layout
- Category filtering (India, Japan, China)
- Interactive lightbox
- Previous/Next navigation
- Keyboard shortcuts
- Hover animations
- Mobile-friendly design

#Built With
- HTML
- CSS
- JavaScript
